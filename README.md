# b2x

## Features

### Binary input

- [ ] Parse big endian
- [ ] Parse little endian
- [ ] Parse as signed
- [ ] Parse as float64
- [ ] Choose grouping (2 bits, 4 bits, 8 bits, example)
- [ ] Force grouping to match up (--strict-group)
- [ ] Output as little endian 
- [ ] Output as big endian 
- [ ] Output as decimal
- [ ] Output as hex 
- [ ] Output as utf8
